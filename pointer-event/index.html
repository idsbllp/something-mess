pointer event

pointer event 主要是移动端手机在使用, 在 PC 上，按一下一个键会触发一次 pointerdown, 按下第二个键不会触发，因为PC 只有一个鼠标。

当移开绑定事件的元素后松开按键，这时不会触发 pointerup, 这时你可能需要使用到 ‘指针捕捉’，即在 pointerdown 的时候调用 `container.setPointerCapture(event.pointerId)`

mouseover vs mouseenter

mouseover 在鼠标进入绑定事件的元素或者其子元素都会触发，默认会冒泡，可以用 stopPropagation 阻止冒泡

mouseenter 指挥在鼠标进入绑定事件的元素触发，不会冒泡
